Como Criar uma Fechadura: Tutorial Completo

Passo a Passo do Projeto

  1. Materiais necessários:
    • Arduino Uno
    • Fechadura elétrica
    • Relé
    • Fonte de alimentação
    • Fios jumper
    • Sensor de proximidade (opcional)
    • Resistor de 10k ohms
  2. Montagem do circuito: Conecte o Arduino ao relé, a fechadura elétrica e outros componentes conforme o diagrama de circuito.
  3. Escrevendo o código: No Arduino IDE, escreva o código para controlar a fechadura, incluindo o uso do relé para acionar a abertura e fechamento.
  4. Testando o sistema: Com tudo montado, teste o funcionamento da fechadura, verificando se ela abre e fecha corretamente.
  5. Melhorias (opcional): Adicione funcionalidades como controle via Bluetooth ou smartphone para maior segurança.

Código Fonte Utilizado


          // Código Arduino para controle de fechadura

          #include   // Inclui a biblioteca Servo para controlar o servo motor

          Servo myServo;  // Cria um objeto "myServo" da classe Servo, que será usado para controlar o servo
          int buttonState = 0;  // Variável para armazenar o estado atual do botão (0 - não pressionado, 1 - pressionado)
          int lastButtonState = 0;  // Variável para armazenar o estado anterior do botão, usada para detectar mudanças de estado
            
          void setup() {
              myServo.attach(9);  // Conecta o servo ao pino digital 9 do Arduino (pino de controle do servo)
              pinMode(10, INPUT);  // Configura o pino 10 como uma entrada (onde o botão está conectado)
              myServo.write(0);  // Define a posição inicial do servo como 0 graus (extremo esquerdo)
          }
            
          void loop() {
              buttonState = digitalRead(10);  // Lê o estado do botão (HIGH se pressionado, LOW se não pressionado)
            
                // Verifica se o botão foi pressionado (estado HIGH)
          if (buttonState == HIGH) {
              myServo.write(90);  // Move o servo para 90 graus (meia posição)
              delay(5000);  // Aguarda 5 segundos (5000 milissegundos) enquanto o servo está na posição de 90 graus
              myServo.write(0);  // Move o servo de volta para 0 graus (extremo esquerdo)
                }
            }
